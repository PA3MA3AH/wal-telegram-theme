Скрипт для автоматической генерации и динамического обновления тем оформления **Telegram Desktop** и **AyuGram** на основе цветовой палитры, созданной утилитой `pywal`.

После первой настройки тема связывается с мессенджером напрямую: при изменении обоев рабочего стола цвета в Telegram меняются **автоматически на лету** без необходимости перезапускать приложение или заново импортировать файл.

---

## Особенности

* **Полная интеграция с Pywal:** Автоматически подтягивает цвета из `.cache/wal/colors.json`.
* **Динамический фон:** Использует ваши текущие обои в качестве бэкграунда чатов (поддерживает `.png`, `.jpg`, и умеет конвертировать первый кадр `.gif`).
* **Обновление «на лету»:** Модифицирует файлы напрямую в папках `tdata` ваших клиентов.
* **Умный сборщик:** Сам упаковывает палитру и изображение в валидный архив формата `.tdesktop-theme`.

---

## Зависимости

Перед использованием убедитесь, что у вас установлены:

1.  **Python 3** (с предустановленным модулем `zipfile`).
2.  **pywal** (для генерации палитры).
3.  **ImageMagick** *(опционально)* — нужен только если вы используете анимированные `.gif` обои для конвертации фона.

## Инструкция по первому запуску

> ⚠️ **Важно:** Инструкцию ниже нужно выполнить **только один раз** при самой первой установке темы. В последующие разы всё будет происходить автоматически.

1. Убедитесь, что у вас сгенерированы цвета в pywal (например, запущен `wal -i /path/to/wallpaper`).
2. Запустите скрипт:
```bash
./wal-ayugram-theme.sh

```


3. Скрипт создаст файл темы. Перейдите в папку `~/.local/share/ayugram-theme/`.
4. Возьмите созданный файл `wal.tdesktop-theme` и **отправьте его самому себе в Избранное (Saved Messages)** в Telegram/AyuGram (можно просто перетащить мышкой в чат).
5. Кликните по отправленному файлу внутри мессенджера и нажмите **«Применить тему»** (Apply theme).

🎉 **Готово!** Теперь связь установлена. При каждом следующем вызове скрипта цвета в Telegram/AyuGram будут меняться мгновенно сами.

## Автоматизация (Интеграция с генерацией обоев)

Чтобы тема обновлялась автоматически каждый раз, когда вы меняете обои через `pywal`, просто добавьте вызов скрипта в ваш основной конфигурационный файл (например, в скрипт смены обоев или конфиг вашего оконного менеджера `i3`/`Hyprland` после вызова `wal`):

```bash
# Пример в вашем скрипте смены обоев:
wal -i "$1"
~/.local/bin/wal-ayugram-theme.sh # Укажите ваш путь к скрипту

```
